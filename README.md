Небольшое iOS-приложение для управления чеклистами — списками задач с возможностью установки напоминаний и иконок для каждого списка.


## Краткий обзор
- Язык: Swift (UIKit)
- Минимальная версия iOS: iOS 13+ (в проекте используется SceneDelegate)
- Главные возможности:
  - Создание/редактирование/удаление чеклистов (списков).
  - Создание/редактирование/удаление элементов в списках.
  - Установка даты/времени и локальное напоминание (UNUserNotificationCenter).
  - Выбор иконки для списка.
  - Сохранение данных в plist в каталоге Documents (PropertyListEncoder/Decoder).
  - Память выбранного открытого списка между запусками.

## Быстрый старт (для разработчика)
1. Клонируйте репозиторий:
   - git clone https://github.com/khamzaev/Checklists.git
2. Откройте проект в Xcode:
   - Откройте файл проекта/воркспейса (обычно Checklists.xcodeproj) в Xcode.
3. Выберите таргет и устройство (симулятор или реальное устройство).
4. Соберите и запустите (Cmd+R).

Примечание: приложение использует локальные уведомления — при первой попытке установить напоминание система запросит разрешение на отправку уведомлений.

## Структура проекта (ключевые файлы и их назначение)
- Checklists/Resources/
  - AppDelegate.swift — настройка делегата уведомлений (UNUserNotificationCenterDelegate).
  - SceneDelegate.swift — создание DataModel и передача в корневой контроллер; сохранение данных при переходе в фон.
- Checklists/DataModels/
  - DataModel.swift — центральный менеджер данных:
    - Загрузка/сохранение массивов Checklist в `Checklist.plist` в Documents.
    - Регистрация значений по умолчанию (`ChecklistIndex`, `FirstTime`).
    - Генерация уникальных ID для элементов чеклиста (`ChecklistItemID` в UserDefaults).
  - Checklist.swift — модель чеклиста (имя, иконка, массив items).
  - ChecklistItem.swift — модель элемента (текст, checked, dueDate, shouldRemind, itemID), планирование локальных уведомлений.
- Checklists/Controllers/
  - AllListViewController.swift — список всех чеклистов (главный экран), управление навигацией, добавление/редактирование списков.
  - ChecklistViewController.swift — просмотр элементов конкретного чеклиста, отметка выполненных, навигация на экран добавления/редактирования элемента.
  - ItemDetailViewController.swift — экран добавления/редактирования элемента (дата, напоминание).
  - ListDetailViewController.swift — экран добавления/редактирования чеклиста (имя, иконка).
  - IconPickerViewController.swift — выбор иконки для чеклиста.
- Ресурсы UI:
  - Storyboard / xib и ассеты (иконки) — для ячеек и экрана выбора иконки.

## Хранение данных
- Списки сохраняются в файл Checklist.plist в каталоге Documents приложения.
- Кодирование/декодирование выполняется через PropertyListEncoder/PropertyListDecoder.
- Индекс выбранного списка хранится в UserDefaults под ключом `"ChecklistIndex"`.
- Флаг первого запуска — `"FirstTime"`.
- Для уникальных itemID используется ключ `"ChecklistItemID"` в UserDefaults.

## Как пользоваться приложением (для конечного пользователя)
1. На главном экране создайте новый чеклист — задайте имя и иконку.
2. Откройте чеклист — добавьте элементы, при необходимости включите "Напомнить" и задайте дату/время.
3. Отмечайте элементы как выполненные — счётчик не выполненных элементов отображается рядом с каждым списком.
4. Данные сохраняются автоматически при переходе приложения в фон.
